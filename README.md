# Rota Unibeer App

Sistema de Gestão Inteligente de Rotas e Avaliação de Performance de Pontos de Venda (PDV).

## Stack Principal
- React + Vite
- TypeScript
- Tailwind CSS + Shadcn UI
- Supabase (Backend/Database/Auth)
- Recharts (Gráficos)
- XLSX (Parsing de planilhas)

## Comandos

```sh
# Instalar dependências
npm install

# Rodar servidor de desenvolvimento
npm run dev

# Build para produção
npm run build
```
